### Overview

This project is a web-based Russian Roulette game developed with a modern front-end stack. The focus is on clean architecture, smooth animations, and an efficient development workflow.

### Technology Stack

* **React 19**
* **TypeScript**
* **Vite**
* **Tailwind CSS 4**
* **Framer Motion**

### Getting Started

1. **Clone the repository**
2. **Install dependencies**

   ```bash
   npm install
   ```
3. **Run the development server**

   ```bash
   npm run dev
   ```
